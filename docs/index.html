源码安装
在 VS Code 中打开源码目录。
安装 vsce，代码如下：

$ npm install -g vsce

打包 .vsix 文件，代码如下：

$ vsce package

在VSCode中右键点击 .vsix 文件即可安装扩展。
软件源安装

jquery4-code-snippet-simplified-chinese可以通过NPM软件源安装。

通过NPM软件源安装jquery4-code-snippet-simplified-chinese，代码如下：

$ npm i jquery4-code-snippet-simplified-chinese
